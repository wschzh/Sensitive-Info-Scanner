## 大模型 API 设计原则

- 默认关闭，用户显式配置 API Key 后才可使用。
- 不上传完整文件，只上传文件路径、规则类型、风险级别、脱敏命中值、少量上下文和统计信息。
- 模型只输出建议和理由，不直接删除文件。
- 删除仍必须经过当前白名单校验和用户二次确认。
- API Key 不默认写入磁盘；如需保存，优先使用浏览器本地配置并允许用户清除。
